## 如何添加文件

1. 将 PDF 或其他文件放入此目录
2. 确保文件名与网页中链接一致
3. 文件会自动提供下载

## 权限设置

```bash
# 设置文件权限（可选）
chmod 644 /home/admin/.openclaw/workspace/website/downloads/*
```

## 注意事项

- 支持任意文件格式（PDF、ZIP、DOC 等）
- 建议文件大小不超过 50MB
- 文件名建议使用英文，避免中文乱码
