## 📌 原文摘要

微软官方在 GitHub 上发布了「AI Agents for Beginners」课程，包含 12 节 Jupyter Notebook，涵盖从基础概念到生产部署的完整学习路径。课程内容包括：AI Agent 架构模式（ReAct、Plan-and-Execute、Chief-of-Agent 等）、多智能体协作、长期记忆、RAG 检索增强、工具调用、Human-in-the-Loop 审批流程，以及 Azure AI Foundry 部署实战。每个 Lesson 都是可运行的 Notebook，配有真实代码和详细注释，适合零基础入门 AI Agent 开发的工程师。

## 🧠 核心要点

* **12 节系统课程**：从概念到部署，覆盖 AI Agent 开发全链路
* **Jupyter Notebook 驱动**：每课都可直接运行，理论与代码结合
* **多种 Agent 架构模式**：ReAct、Plan-and-Execute、Chief-of-Agent 等主流范式都有覆盖
* **多智能体协作**：讲解多个 Agent 之间的任务分配与通信机制
* **记忆与 RAG**：长期记忆、向量检索增强生成等实战主题
* **Azure AI Foundry 部署**：生产环境部署有完整指导
* **Microsoft 官方背书**：内容质量和可持续性有保障

## 🔥 推荐理由

市面上 AI Agent 的教程碎片化严重，要么是概念科普，要么是某个具体框架的 Hello World。微软这套课程难得地做到了系统性和实战性并重——12 个 Notebook 覆盖从单 Agent 执行循环到多 Agent 协作的完整知识图谱，且都是可运行的真实代码。对于想从传统 LLM 调用进阶到 Agent 系统设计的工程师，这是一份高质量的入门路径。配合 Azure AI Foundry 的部署实战，学习完成后可以直接输出可工作的原型。
