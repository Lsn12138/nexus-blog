史上最快破5万星项目：Claude Code开源替代登顶GitHub

AI开发工具Rust

原文摘要

instructkr/claw-code 是一个重构自泄露Claude Code源码的项目，2026年3月31日发布，仅用2小时便突破5万star，刷新GitHub史上最快纪录。目前星标数已超过5.6万，fork数近6万，全部为 Rust 重写。它不仅是源码存档，更致力于打造真正可用的AI编程工具，提供比闭源版更强大的实际功能。

核心要点

  • 2小时破5万star，创GitHub历史纪录，超越此前所有项目
  • 全部代码用 Rust 重写，性能与安全性大幅提升
  • 保留并改进了泄露版的核心能力，同时新增自研功能
  • 开源协议存疑（泄露源码），但项目方明确表示将探索合法开源路径
  • 成为Claude Code最知名的开源替代方案，引发社区广泛讨论

推荐理由

这不是一个简单的"搬运项目"。虽然发版时机借助了泄露代码的流量，但团队选择全面Rust重写、持续迭代新功能，本身已经超越了存档的范畴。2小时破5万star的现象级传播，背后是整个开发者社区对AI编程工具开源替代的强烈渴望。同时，这个项目也再次把"AI编程工具开源化"这个命题推到台前——当 Claude Code 有了开源版，Cursor、Warp、Copilot 又该如何应对？

热度信息

GitHub Trending 全榜第一，星标 56,000+，fork 近 60,000（截至2026年4月1日）。Hacker News 相关讨论帖登顶，评论超过500条，多平台同时引爆。
