## 📌 原文摘要

DeepTutor 是香港大学数据科学实验室推出的 Agent 原生个性化学习平台，刚发布 v1.0.0 重大版本。它不是简单的聊天机器人，而是一个拥有独立记忆、人格和技能集的自主导师系统。支持多模式统一工作区（对话、深度解答、Quiz 生成、深度研究、数学动画），每个 TutorBot 都有持久记忆。它还内置 AI 协作者 Co-Writer、引导式学习路径、知识库管理等功能，采用 TutorBot 多渠道机器人 + 插件化工具架构（Tools + Capabilities 两层）。

## 🧠 核心要点

* **Agent 原生架构 v2.0**：从底层重建，采用两层插件模型（Tools + Capabilities）
* **TutorBot 个性化导师**：每个 Bot 有独立工作区、记忆和技能集，可自主设置提醒和学习进度
* **统一工作区五大模式**：Chat、Deep Solve、Quiz Generation、Deep Research、Math Animator 共享上下文
* **AI Co-Writer 协作编辑器**：Markdown 编辑器中 AI 作为一等公民，可辅助写作润色和扩写
* **引导式学习路径**：将资料自动转化为多步骤视觉学习旅程，对每个知识点进行深入讨论

## 🔥 推荐理由

DeepTutor 解决了一个真实痛点：现有 AI 辅导工具大多是通用聊天机器人，无法记住学生的学习进度和偏好。DeepTutor 的 TutorBot 真正实现了"个性化"——它会在整个学习旅程中持续追踪学生的状态，并根据反馈调整教学策略。v1.0.0 的 Agent 原生架构意味着系统可以灵活扩展，未来接入更多工具和能力。值得关注的是，它采用了 nanobot 作为 Bot 底层框架，这对于研究 Agent 系统架构的同学有参考价值。
