## 📌 原文摘要

OpenScreen 是一款开源免费的录屏工具，旨在替代付费的 Screen Studio，为用户提供无订阅、无水印、免费商用的产品演示录制方案。支持全屏或指定窗口录制、自动缩放、手动缩放、麦克风/系统音频录制、自定义背景、运动模糊、视频裁剪、加速减速、多种分辨率导出等功能。基于 Electron + React + TypeScript + Vite + PixiJS 构建。

## 🧠 核心要点

* **完全免费**：无订阅、无水印，商用亦免费（MIT 协议）
* **自动缩放 + 运动模糊**：呈现类似 Screen Studio 的专业演示效果
* **跨平台**：支持 macOS（13+）、Windows、Linux（含 AppImage）
* **系统音频捕获**：支持 macOS 14.2+ 音频捕获、Linux PipeWire、Windows 开箱即用
* **编辑能力**：裁剪、变速、多比例导出、配音注释

## 🔥 推荐理由

对于需要制作技术视频、API 演示、产品 Demo 的开发者，OpenScreen 提供了一个零成本的高质量替代方案。Screen Studio 每月 $29 的订阅费让许多个人开发者和小型团队望而却步，OpenScreen 以开源方式填补了这一空白。其自动缩放和运动模糊功能尤其适合需要引导观众注意力的演示场景。不过当前版本处于 Beta，稳定性有待验证。
